## How to run
1. Download respectives files from the intra subject (you need to unzip .tgz. You can use tar command).
2. Then unziped, run ./run.sh or iniciate index.html on a web browser. (Chrome, Opera supports correctly).
3. In the index page, select Training mode write your intra login and press "start!".
